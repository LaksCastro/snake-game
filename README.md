# How to clone project

- To clone this repo

```
git clone https://github.com/LaksCastro/snake-game.git
```

- To change path

```
cd snake-game
```

- To install dependencies

```
yarn install
```
or, for npm users:
```
npm install
```

- To run development server

```
yarn dev
```
or, for npm users:  
```
npm run dev
```

- To create static files

```
yarn pre-build
npm run pre-build
```

- To deploy to github pages

```
yarn build
npm run build
```
